# Libreboot-style flash workflow (lab-safe template)

This file is a template for documenting a controlled flash workflow for supported hardware.

## Suggested checklist

1. Confirm the target device and supported hardware matrix.
2. Back up the current firmware image and all relevant firmware variables.
3. Record the current BIOS/firmware version and boot state.
4. Verify that the target flash tool and image are supported for the exact platform.
5. Prepare a rollback plan and recovery method.
6. Perform the flash only in a controlled lab environment.
7. Verify the resulting firmware state after the operation.

## Notes

- This template is for documentation only.
- It does not replace official vendor guidance or signed firmware processes.
- It should not be used to bypass platform security or vendor restrictions.
