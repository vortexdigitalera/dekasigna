# Libreboot lab notes

This folder provides a lab-safe, documentation-oriented scaffold for exploring Libreboot-style firmware workflows on supported hardware.

## Scope

The content here is intended for:

- firmware research and lab documentation
- safe backup and recovery planning
- flash tool and installer workflow research
- compatibility notes for supported devices

## Safety

- Do not flash firmware on a production or mission-critical device without verified vendor support and a rollback plan.
- Always preserve a full firmware backup before attempting any update or flash procedure.
- Use only firmware images and tools that are explicitly supported for the target device.
- This repository does not provide a firmware payload, signed image, or vendor-approved flash path.
